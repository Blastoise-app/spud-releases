# Spud

Spud is a menu-bar workspace manager for macOS: capture your window layout —
including browser tabs — and restore it later with one click.

## Download

**[⬇ Download Spud (latest)](https://github.com/Blastoise-app/spud-releases/releases/latest/download/spud.dmg)**

Open the DMG and drag Spud to Applications. On first launch, grant
Accessibility permission when prompted — Spud needs it to read and move other
apps' windows.

Every release is signed (Developer ID: Blastoise, LLC), notarized by Apple,
and stapled, so macOS opens it with no security warnings.

## Requirements

- macOS (Apple silicon or Intel)
- Browser tab capture supports Safari, Chrome, Brave, Edge, Arc, and other
  Chromium-based browsers
